# @clsc/ui

Shared React UI component library based on Semi Design.

## Responsibilities

- Provide reusable, accessible React components.
- Wrap `@douyinfe/semi-ui` primitives where appropriate.

## Allowed dependencies

- `react`, `react-dom`, `@douyinfe/semi-ui`, testing utilities.

## Forbidden

- Direct backend calls, business logic, global state.

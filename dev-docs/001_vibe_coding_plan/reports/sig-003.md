# SIG-003 全量 reconciliation 报告

## 任务目标

定义全量 reconciliation 契约：校验、原子切换、缺失设备保留窗口、有界事件缓存。

## 变更摘要

- `crates/signaling-adapter/src/reconciler.rs`：新增 `ReconciliationCursor::validate` 方法，`InboundEvent`、`ReconciliationOptions`、`ReconciliationReport`，以及 `SignalingReconciler::reconcile_full`。
- `reconcile_full` 校验 `validate_before_switch`、`missing_window_seconds`、`bounded_cache_size` 与 cursor limit 后返回 `Unsupported`。

## 验证

- `cargo clippy -p signaling-adapter --all-targets -- -D warnings` 通过。
- 新增单元测试覆盖 `reconcile` 返回 `Unsupported`、`reconcile_full` 校验通过返回 `Unsupported`、无效 cursor 与零缓存报错。

## 未完成/降级

- 真实上游 signaling 抓取与投影原子切换需要 Phase 3 的 JetStream/NATS 运行时，当前显式返回 `Unsupported`。

# APP-001 应用用例模板报告

## 任务目标

定义应用层统一用例模板与 ports，确保写操作遵循验证、鉴权、领域变更、持久化、审计的顺序。

## 变更摘要

- `crates/application/src/usecase.rs`：定义 `UseCase`、`WriteRequest<T>`、`WriteResponse<T>`、鉴权/审计/截止时间的通用 helper。
- `crates/application/src/{tenant,user,organization,role,device,config}.rs`：各 aggregate 的 application port 与默认服务。

## 验证

- `cargo clippy -p application --all-targets -- -D warnings` 通过（`--workspace` 本地通过）。
- 应用层 fake port 测试覆盖成功、拒绝、冲突、取消、审计失败。

## 未完成/降级

- `UnitOfWork` 已在 `storage-api`/`storage-postgres` 实现；应用服务当前直接调用 repository，但 `WriteRequest` 已预留 idempotency 字段，`PostgresUnitOfWork` 可直接接入后续重构。

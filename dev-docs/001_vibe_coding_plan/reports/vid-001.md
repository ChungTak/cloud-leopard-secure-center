# VID-001 PlaybackEntitlement 报告

## 任务目标

定义播放授权领域模型，将 token 绑定到 tenant/principal/camera/session/protocol，且不暴露上游 URL。

## 变更摘要

- `crates/domain-media/src/lib.rs`：新增 `MediaSession`、`MediaToken`、`PlayerPolicy`；扩展 `PlaybackEntitlement` 与 `CreateEntitlementRequest`。
- `MediaToken` 携带绑定元数据，不保存 URL；`MediaSession` 表示 signaling session。

## 验证

- `cargo clippy -p domain-media --all-targets -- -D warnings` 通过。
- `domain-media` 单元测试 `UnsupportedMediaPort` 返回 `Unsupported`。

## 未完成/降级

- 真实 media session 状态机由上游 media engine 提供；当前 `UnsupportedMediaPort` 返回 `Unsupported`。

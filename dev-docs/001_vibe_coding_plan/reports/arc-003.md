# ARC-003 角色化装配报告

## 任务目标

定义 `Role` 与角色化装配的生命周期顺序，为单进程 `All` 角色和多角色集群二进制提供装配骨架。

## 变更摘要

- `crates/cluster-adapter/src/lib.rs`：新增 `Role` 枚举（`Api`/`Workflow`/`Projection`/`Scheduler`/`PluginHost`/`All`）与 `Role::expand`。
- `crates/cluster-adapter/src/assembly.rs`：新增 `LifecyclePhase`、`Lifecycle`（固定启动/关闭顺序）和 `ClusterAssembler::validate_lifecycle`。

## 验证

- `cargo clippy -p cluster-adapter --all-targets -- -D warnings` 通过。
- `cargo test -p cluster-adapter` 通过。

## 未完成/降级

- 真实多二进制装配、NATS KV 租约、滚动 drain 等需要运行时基础设施，继续返回 `Unsupported`/`Unavailable`。

# AUD-003 保留与清理报告

## 任务目标

定义保留策略、legal hold、租约化清理与分区 drop 前的审计/备份确认。

## 变更摘要

- `domain-audit/src/retention.rs`：`RetentionTarget`、`RetentionPolicy`、`TenantRetentionOverride`、`LegalHold`、`CleanupLease`、`CleanupBatchResult`。
- `storage-api/src/lib.rs`：`RetentionRepository` trait，含 `save_policy`、`get_policy`、`set_tenant_override`、`get_effective_days`、legal hold、lease、batch cleanup、以及新增 `drop_partition`。
- `storage-postgres/src/retention_repository.rs`：Postgres 实现；`drop_partition` 无备份确认返回 `Invalid`，有确认时写审计记录并返回 `Unsupported`。

## 验证

- `cargo clippy -p storage-postgres --all-targets -- -D warnings` 通过。
- 完整集成测试需要 PostgreSQL 容器；未运行 `nextest`。

## 未完成/降级
